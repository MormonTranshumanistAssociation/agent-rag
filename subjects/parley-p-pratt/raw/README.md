# Raw-source workspace for Parley P. Pratt

This directory is for source acquisition and review artifacts that should **not** be treated as clean retrieval corpus text.

## Layout

```text
raw/
  scans/   # downloaded page images, PDFs, or source captures
  ocr/     # raw OCR or extracted text before normalization
  review/  # OCR review packets generated by `agent-rag prepare-ocr`
```

## Workflow

1. Put the original scan/PDF/source capture in `raw/scans/`.
2. Put raw OCR or extracted text in `raw/ocr/`.
3. Run `agent-rag prepare-ocr ... --output-dir raw/review/<document-id>`.
4. Proofread `candidate.md` using the generated `proofread_prompt.md`.
5. Move the reviewed document into `clean/primary/`, `clean/secondary/`, or `clean/context/` as appropriate.

## Important rule

Nothing in `raw/` is automatically trustworthy enough for retrieval. It becomes corpus text only after review and explicit placement into `clean/`.
